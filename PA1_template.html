activity <- read.csv(unz(temp, "activity.csv"))

activity$date <- ymd(activity$date)

What is mean total number of steps taken per day?

Using dplyr, a dataframe steps_per_day was generated that sums total steps across all time intervals for a given day.

steps_per_day <-
    activity %>%
        group_by(date) %>%
        summarise(total = sum(steps, na.rm = TRUE))

The steps_per_day dataframe was used to produce the following histogram:

hist(steps_per_day$total, breaks = 11, main = "Total steps per day", 
     xlab = "steps per day")

mean_per_day <- mean(steps_per_day$total)
median_per_day <- median(steps_per_day$total)

From this data, the mean number of steps per day is 9354.2295, and the median steps per day is 1.0395 104.

What is the average daily activity pattern?

steps_per_interval <-

Imputing missing values

The number of NA values in the steps column of the activity dataframe were calculated using the following:

missing <- is.na(activity$steps) %>% which %>% length

The number of missing values is 2304.

I have decided to replace these missing values with the median number of steps per interval from the steps_per_interval dataframe.

replacement <- median(steps_per_interval$total)

The value being used for the imputation is 34.1132 steps.

Using dplyr, the activity data frame was regenerated with all NA values substituted with the replacement value of 34.1132 steps.

imputed_activity <-
    activity %>%
        group_by(date, interval) %>%
        mutate(steps = ifelse(is.na(steps),
                replacement,
                steps)
               ) %>%
        as.data.frame

dplyr was then used again to generate the imp_steps_per_day dataframe which sums the total number of steps for each interval across each day. This dataframe can then be used to determine the mean and median of the imputed dataset.

imp_steps_per_day <-
        imputed_activity %>%
            group_by(date) %>%
            summarise(total = sum(steps, na.rm = TRUE))
imp_mean_per_day <- mean(imp_steps_per_day$total)
imp_median_per_day <- median(imp_steps_per_day$total)

The mean from the imputed data is 1.0643 104 and the median is 1.0395 104. The median is identical to that of the non-imputed dataset. The mean of the imputed dataset, however, has increased its value by ~14%.

A histogram of the data largely maintains the shape of the distribution. However, it is clear that the large number of zeroes in the original dataset have now been replaced with 34.1132 in the imputed dataset.

hist(imp_steps_per_day$total, breaks = 11, main = "Total steps per day", 
     xlab = "steps per day")

Are there differences in activity patterns between weekdays and weekends?

Using dplyr, a new dataframe, wkds, was generated that is identical to the imputed data imputed_activity, except it contains an additional factor column listing each date as either a Weekday" or Weekend. This was done by first generating the day column using the weekdays() function, followed by grep-substitution of all day names to the appropriate Weekend or Weekday substitution.

wkds <- 
    imputed_activity %>%
        group_by(date, interval) %>%
        summarise(day = weekdays(date), steps) %>%
        as.data.frame

wkds <- sapply(wkds, gsub, pattern = '(Saturday)|(Sunday)', 
               replacement = 'Weekend') %>%
                as.data.frame

wkds <- sapply(wkds, gsub, pattern = "^.*day$", 
                replacement = 'Weekday') %>%
                as.data.frame
wkds$day <- wkds$day %>% as.factor
wkds$steps <- as.numeric(wkds$steps)

To determine the steps per interval pattern across weekends and weekdays, the wkds was run through a dplyr pipe to generate the dataframe steps_per_day_interval which determines the mean number of steps for each interval across all days in the dataset while maintaining the weekend/weekday designations as factors.

steps_per_day_interval <-
    wkds %>%
    group_by(interval, day) %>%
    summarise(total = mean(steps, na.rm = TRUE))

steps_per_day_interval$interval <- steps_per_day_interval$interval %>% as.numeric

A faceted timeseries plot of weekend and weekday step activity was produced using ggplot2. The activity patterns are very similar across weekends and weekdays, and both resemble the pattern from the original, non-imputed dataset.

ggplot(steps_per_day_interval, aes(interval, total)) + geom_line() +
           facet_grid(day ~ .) + xlab("time interval") + ylab("mean number of steps")
